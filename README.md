# VoyagerEngine
A modern C++ game engine focused on learning graphics programming and engine architecture.

## Vision
The goal of Sophia is not to compete with existing game engines, but to understand how they are built.

Every system is designed and implemented from scratch whenever it provides educational value, while external libraries are only used when they are outside the scope of the project.

## Goals
- Understand game engine architecture
- Learn real-time rendering
- Build a modular and maintainable codebase
- Document technical decisions
